## 🚀 クイックスタート

Claude Code CLI と Fluorite MCP を今すぐ始めましょう。

### 前提条件
- **Node.js**: 18.0以上（最適なパフォーマンスには20.0以上を推奨）
- **Claude Code CLI**: MCP対応の最新版
- **npm**: 8.0以上または互換パッケージマネージャー

### インストール

**クイックセットアップ**（2分インストール）:
```bash
# 1. Fluorite MCPをグローバルインストール
npm install -g fluorite-mcp

# 2. Claude Code CLIに追加
claude mcp add fluorite-mcp -- fluorite-mcp

# 3. インストール確認
claude mcp list
```

詳細なインストール手順については、[インストールガイド](./installation.md)をご参照ください。
