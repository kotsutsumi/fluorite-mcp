# ライブラリ仕様カタログ

Fluorite MCPが提供するライブラリ仕様の一覧です。これらの仕様はClaude Code CLIにコンテキストを提供し、より正確で実用的なコード生成を可能にします。

::: info
仕様ファイルは `src/catalog/` ディレクトリにYAML形式で保管されています。
:::
